# ぴこっち 🥚

昔なつかしの携帯育成ゲーム風 **PWA(プログレッシブ・ウェブ・アプリ)** です。
ブラウザだけで動き、iPhone のホーム画面に追加するとアプリのように遊べます。

- たまごから育てて、お世話の質で進化先が変わる(3系統)
- ごはん・おやつ・ミニゲーム・そうじ・くすり・でんき・しつけ
- 夜は寝る/放置すると病気になる/最期はお別れ → 次の世代へ
- アプリを閉じている間も時間が進む(開くと「留守中ダイジェスト」を表示)
- きせかえ: **レトロ液晶** ⇔ **モダンかわいい** の2テーマ
- 通信不要・完全オフライン動作(セーブはブラウザ内 localStorage)

## 遊べるようにする手順(初回だけ)

1. GitHub のこのリポジトリで **Settings → Pages** を開く
2. 「Build and deployment」の Source を **Deploy from a branch** にして、
   Branch に **main / (root)** を選んで Save
3. 数分待つと `https://<ユーザー名>.github.io/tamagotchi/` で公開される

## iPhone にインストールする手順

1. iPhone の **Safari** で上の URL を開く
2. 共有ボタン(□に↑)→ **「ホーム画面に追加」** をタップ
3. ホーム画面の「ぴこっち」アイコンから起動(全画面のアプリとして動きます)

> プッシュ通知には対応していません(サーバーが必要なため)。
> かわりにアプリを開いたとき、留守中に起きたことをまとめて教えてくれます。

## あそびかた

| アイコン | 意味 |
|---|---|
| 🍙 ごはん | 食事(おなか+1)/ おやつ(ごきげん+1、太りやすい) |
| 💡 でんき | 消灯・点灯。寝たら電気を消してあげよう |
| 🎮 あそぶ | 左右当てゲーム。勝つとごきげんアップ、体重も減る |
| 💊 くすり | 病気(どくろマーク)のときに |
| 🧹 そうじ | うんちのお掃除。放置すると病気に |
| 📊 メーター | おなか・ごきげん・しつけ・体重などの確認 |
| 📣 しつけ | 「よんでる」のに何も要求がないときはしつけのチャンス |
| ❗ よんでる | お世話が必要なサイン |

お世話ミスが少なく、しつけをしっかりすると良い子に進化します。

## 開発メモ

フレームワーク・ビルド工程なしの Vanilla JS (ES Modules)。push すればそのまま GitHub Pages に反映されます。

```
index.html            画面(筐体UI)
css/style.css         テーマ(レトロ/モダン)
js/engine.js          ゲームロジック(DOM非依存の純粋ロジック)
js/main.js            画面制御・入力・ゲームループ
js/render.js          Canvas ドット絵描画
js/sprites.js         キャラのドット絵データとパレット
js/storage.js         セーブ(localStorage)
js/sound.js           ピコピコ音(WebAudio)
js/minigame.js        左右当てゲーム
sw.js                 Service Worker(オフラインキャッシュ)
manifest.webmanifest  PWA マニフェスト
test/engine.test.js   ロジックのユニットテスト
```

- テスト: `node --test test/engine.test.js`
- ローカル起動: `python3 -m http.server 8000` → `http://localhost:8000`
- **リリース時の注意**: ファイルを変更したら `sw.js` の `CACHE_VERSION` を上げると、
  インストール済み端末のキャッシュが確実に更新されます
